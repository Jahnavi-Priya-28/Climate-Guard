# Climate Guard 🌍

Climate Guard is an Augmented Reality (AR) based environmental awareness application designed to educate users about climate change and environmental conservation through immersive 3D experiences.

## Overview

The project uses AR technology and 3D models to visualize environmental concepts, helping users better understand climate-related challenges and sustainable solutions.

## Features

* 🌱 Interactive Augmented Reality experience
* 🏞️ 3D environmental visualization
* 📱 Web-based AR application
* 🌍 Climate awareness and education
* 🎯 User-friendly interface
* 🚀 Lightweight and accessible

## Technologies Used

* HTML5
* JavaScript
* WebXR / AR Technologies
* GLB 3D Models
* CSS3

## Installation

### Clone the Repository

```bash
git clone https://github.com/Jahnavi-Priya-28/Climate-Guard.git
cd Climate-Guard
```

### Run Locally

Open:

```text
index.html
```

in your browser or use a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Usage

1. Launch the application.
2. Grant camera permissions when prompted.
3. Scan the environment.
4. Interact with the 3D climate models.
5. Explore environmental awareness content through AR visualization.
